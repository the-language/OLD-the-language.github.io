## 2. 誤

### 2.1. 停機

實現應該識別出儘量多的停機，改爲`(誤 (構 {引 誤/界/停} ()))`。

### 2.2. 內置

內置的機產生的`誤`是`(誤 (構 {引 誤/界} (列 -名 -式 -其他)))`。`-名`是它的名稱，`-式`是它的參數。

內置特殊形式的`誤`是`(誤 (構 {引 誤/界/式} (列 -名 -境 -式
-其他)))`。`-名`是它的名稱，`-境`是環境，`-式`是它的參數。
